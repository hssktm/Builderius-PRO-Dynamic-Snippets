View Posts Filtered by Category Slug
This snippet fetches a list of posts filtered by a specific category.

How to edit the code
Locate the following sections to customize the snippet:

Change categories: Look for the line terms: ["slug-1", "slug-2"] and replace the values with your category slug.

Category structure: The categories: expression_result section allows categories to be displayed as a direct array.

Customization: You can choose to remove the categories block, the images, or even the pagination section if you don't need them in your data output.
